# TCN_Design
## Purpose
The purpose of this project is to implement a temporal convolutional network which is going to be a part of computer networking related project. This implementation will focus more on the areas of DDoS classifying, QoS / QoE enhancing and flow management. 
First implementations will go through the necessary learning curves to understand the behaviour of the TCN models, through implementing relevant designs in mnist classificiation and ECG recognition.
